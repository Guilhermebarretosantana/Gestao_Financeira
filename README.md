# Gestao_Financeira
Aplicação desenvolvida para controle financeiro pessoal, permitindo o gerenciamento de receitas, despesas e saldo mensal de forma pratica e intuitiva. O sistema possibilita o registro de transações, calculo automático de lucro/ prejuízo e visualização de dados através de gráficos interativos, facilitando a análise financeira de decisões.
